# OS setup.sh files

We recommend running install/quickstart.sh instead of the setup.sh files.

The setup.sh files are NOT used with the "install" folder desktop.conf

As a terminal-only alternative, you can use the setup.sh files...

#### Mac OS

Run the [Mac install script](../MacOS) in your local terminal.<!-- Type "sh " than drag the setup.sh file onto the terminal to complete the path.-->
After installing, click "Get Info" on the new apps and unlock in the lower right.  

#### Windows 10
Run the [Windows install script](../Windows) in a local terminal. 

#### Linux <!-- Tested on Ubuntu 18 LTS -->
Run the [Ubuntu install script](../Ubuntu) in a local terminal.
